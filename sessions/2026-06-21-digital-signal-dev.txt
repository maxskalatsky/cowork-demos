─── WHAT WAS BUILT ────────────────────────────────────────────────────────────

Renamed the product from "S&A Digital Audit" to The Digital Signal.

Agent readiness scoring
  - Tried Cloudflare URL Scanner. Token was invalid (auth error), always failed.
  - Replaced with rules-based scorer that fetches signals directly from the site.
  - Baseline 40 (EMERGING floor). Points: llms.txt +20, robots not blocking AI
    crawlers +15, schema markup +10, semantic headings +10, text >= 300 words +5.
  - Cap 100. BASIC only below 40. skalatsky.com scores ADVANCED (80).

Positioning section — full rebuild
  Back end: scorePositioning() runs three independent tests on fetched HTML:
    THE HOOK   → lands / partial / missing
               Strong H1 + specific or emotionally compelling language
    THE FIT    → clear / partial / unclear
               Clear service/product description visible early, no heavy jargon
    THE RELEVANCE → connects / partial / missing
               Second-person language, outcome or persona framing
  Trust killer disqualifier: generic filler in opening 500 chars, 3+ AI buzzwords,
    4+ competing CTAs, or content overload with no subheadings.
  Twelve-verdict library matched to combinations. Color green/yellow/red.
  Returns: { hook, fit, relevance, disqualifier, color, verdict }

  Front end: left border color, three labeled tags (THE HOOK — lands, etc.),
    single TRUST KILLER tag when disqualifier fires, verdict as plain sentence.
  Give First removed entirely.

Email gate + KV rate limiting

  - Worker checks Cloudflare KV namespace AUDITS before running.
  - Same email blocked with: "You have already run a free Signal. Email

  - Every successful audit logged to KV: email, url, timestamp, seoScore,
